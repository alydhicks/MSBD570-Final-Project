# MSBD570-Final-Project
This is for classwork submission documentation.

Because the dataset is too large to load into this repository, you must go to the following website to download: https://physionet.org/content/challenge-2016/1.0.0/ 

Once retrieved, you must load the dataset into your Colab notebook by first creating a folder in the content tab named challenge_heart_sounds, where you will then upload the zipped data file into. The code will then unzip the file and load it to the notebook.
